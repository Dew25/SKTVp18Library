========================
Как запустить программу
========================
----StartSaveToBase----
Для запуска программы, которая будет работать с локальной базой данных MySQL,
необходимо создать ee со следующими параметрами:
databaseName: jktvr18library
user: jktvr18library
password: jktvr18library
collation: utf8.general_ci

----StartSaveToFile----
Этот ярлык запустит программу с сохранением данных в файлах, 
которые будут располагаться в текущей директории.
